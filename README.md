# OSRS Profit Tracker

A command-line tool for tracking real-time profit based on item prices from the OSRS Wiki API. This software allows users to monitor the profitability of item recipes in Old School RuneScape (OSRS) by calculating profits based on the market values of input and output items.

## Features

- Fetch real-time item prices from the OSRS Wiki API.
- Calculate profit based on specified recipes.
- Configurable polling interval to update profit estimates.
- Easy command-line interface for recipe tracking.
